# Showcase Asset Provenance

This notice covers the seven compressed Egypt dogfight images in this directory. They are documentation and demonstration media; they are not included in the packaged Blender extension ZIP and are not covered by the extension source code's `GPL-3.0-or-later` license.

## Origin

- Created as render, viewport, inspection, and playblast exports from Michael's private `egypt.blend` project, working title *The Last Vault*.
- Produced while exercising Blender Agent Bridge against Blender 5.1.2 and the EEVEE renderer.
- Added to this repository by Michael in commit `2bd2620b9355c96a0933ed2ccf51b8ebfc568142` on 2026-06-19.
- The source `.blend`, full-resolution video, and upstream asset archives are not distributed in this repository.

The scene included original arrangement, animation, cameras, lighting, effects, procedural smoke/fire/tracers, and bridge-generated workflow changes. Known external inputs included P-40 and Horten Ho IX aircraft meshes, a Human Base Meshes bundle, and Poly Haven sandstone materials and a desert HDRI. Poly Haven material/HDRI inputs were obtained under CC0. The repository does not retain the upstream URLs, authors, or license records for the aircraft and human mesh inputs, so those inputs cannot currently be independently verified from this repository alone.

## Licensing Boundary

Copyright in Michael's original render composition and bridge-produced documentation exports is retained by Michael. Because the complete upstream provenance for every depicted third-party mesh was not preserved, no standalone reuse or derivative-work license is granted for these showcase files. They should not be treated as stock media, CC0 assets, or GPL-covered source assets.

The files remain in the repository solely to document the bridge's historical test workflow. A future release may replace them with fully procedural or fully provenance-cleared showcase media.
